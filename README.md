# Hex Tower Defense

Tower Defense игра на шестигранной сетке с возможностью игры для двух игроков, против бота или по уровням.

## Особенности

- **Шестигранное поле**: Игровое поле размером 15x45 гексагонов
- **Два типа объектов**: 
  - Башни (прямоугольники) - защищают базу
  - Солдаты (прямоугольники) - атакуют вражескую базу
- **Улучшения**: Башни и солдаты могут быть улучшены
- **Три режима игры**:
  - Два игрока (PvP)
  - Против бота (PvE)
  - По уровням (Campaign)

## Архитектура

Игра реализована с использованием паттерна BLoC (Business Logic Component):
- `GameBloc` - управление состоянием игры
- `PlayerBloc` - управление выбором игрока
- `TowerBloc` - управление башнями
- `SoldierBloc` - управление солдатами

## Запуск

Игра использует ES6 модули, поэтому требуется локальный HTTP сервер.

### Вариант 1: Python (рекомендуется)
```bash
python server.py
```
Затем откройте http://localhost:8000 в браузере

Или используйте двойной клик по `start-server.bat`

### Вариант 2: Python встроенный сервер
```bash
python -m http.server 8000
```

### Вариант 3: Node.js (http-server)
```bash
npx http-server -p 8000
```

### Вариант 4: VS Code Live Server
Установите расширение "Live Server" в VS Code и нажмите "Go Live"

**Важно:** Не открывайте `index.html` напрямую через `file://` - модули ES6 не будут работать!

## Управление

1. Выберите тип башни или солдата на панели управления
2. Кликните на поле для размещения башни или отправки солдата
3. Кликните на башню или солдата и нажмите кнопку улучшения для апгрейда
